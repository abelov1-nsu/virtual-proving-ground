## Инструкция для начала работы

### 1. Требования

- Python 3.13.13  
- Библиотека `pygame`

Установка зависимостей:
```bash
pip install pygame
```
2. Запуск визуализации

Для запуска визуализатора сценариев выполните:
```bash
python scenario_runner_gui.py
```
По умолчанию будут запущены все сценарии из папки scenarios/.

3. Запуск создания сценария

Для запуска редактора сценариев выполните:
```bash
python scenario_creator.py
```
